# Long-Term Investment Methodology

This folder contains the framework and analysis methodology for long-term stock investments focused on enduring businesses and value compounding.

## Framework Stages

The long-term investment methodology follows a systematic three-stage process:

### 1. Discovery (Round 1)
Initial screening and identification of potential investment opportunities.
- File: `1-Discovery.md`

### 2. Stability Check (Round 2)
Assessment of business stability and fundamental strength.
- File: `2-StabilityCheck.md`

### 3. Value Analysis (Round 3)
Deep intrinsic value analysis and buy/no-buy decision framework.
- Main overview: `3-ValueAnalysis.md`
- Key parameters: `3.2-ValueAnalysis-KeyParameters.md`
- Core calculation steps: `3.3-ValueAnalysis-CoreCalculationSteps.md`
- Decision framework: `3.4-ValueAnalysis-BuyNoByDecision.md`

## Philosophy

The long-term methodology emphasizes:
- Compounding businesses over trading opportunities
- Discipline and patience over market timing
- Fundamental value over market sentiment
- Business partnership mindset over speculation

For implementation code and tools, see the `Framework/` directory.
